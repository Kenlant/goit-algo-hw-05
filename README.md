### Детальний аналіз

**Boyer-Moore — найшвидший в усіх тестах.** Перевага — у 4.5× над KMP і у 12× над Rabin-Karp за середнім часом. Алгоритм ефективно пропускає великі ділянки тексту завдяки таблицям зсуву, що особливо помітно на підрядках із звичайними символами природної мови.

**KMP стабільно посередній.** Гарантована лінійна складність O(n + m) робить його передбачуваним, але без пропуску символів він поступається Boyer-Moore на практичних текстах.

**Rabin-Karp найповільніший.** Накладні витрати на хешування роблять його неконкурентним для пошуку одного підрядка. Алгоритм доцільний лише при одночасному пошуку багатьох патернів.

**Вигаданий підрядок повільніший за наявний** для всіх алгоритмів: відсутність збігу змушує сканувати весь текст до кінця. Boyer-Moore уповільнюється у 1.6×, KMP у 1.9×, Rabin-Karp у 2.8× — найбільше страждає алгоритм із хешуванням.

**Розмір тексту впливає лінійно.** article2 (~17 590 символів) дав приблизно у 1.5× вищі часи порівняно з article1 (~12 658 символів), що відповідає очікуваній O(n) залежності.
